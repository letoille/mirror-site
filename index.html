🛡 安全设计 · 绝不碰游戏进程

🛡 Safe by design

魔镜绝不注入、读取或修改游戏进程与内存，也不修改任何游戏文件。它只会：读取游戏日志文件、读取你主动复制的物品文本、以及按识别热键时截取一帧屏幕做本地 OCR。所有查价均通过官方交易行 API，与官方交易网站工作方式一致。

Mirror never injects into, reads, or modifies the game process or memory, and never modifies any game files. It only reads the game's log file, reads item text you copy yourself, and captures one screen frame for local OCR when you press the scan hotkey. All price lookups use the official trade API, exactly like the official trade site.
