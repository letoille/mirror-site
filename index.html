FAQ

常见问题

Frequently asked

使用魔镜会被封号吗？Will using Mirror get me banned?
魔镜只读取剪贴板与游戏日志文件，并调用官方交易行 API，不会注入或修改游戏进程。它的工作方式与官方交易网站一致，属于常见的第三方交易辅助类别。请自行遵守游戏服务条款。 Mirror only reads the clipboard and the game's log file, and calls the official trade API. It never injects into or modifies the game process — it works the same way the official trade site does. Always follow the game's Terms of Service.
安装包从哪里下载？会占用我的仓库吗？Where does the installer come from?
安装包托管在本页面（GitHub Pages）上。即便源码仓库是私有的，Pages 站点及其中的下载文件对所有人公开，源码本身仍然保密。 The installer is hosted on this page (GitHub Pages). Even when the source repository is private, the Pages site and its download files are public while the source code stays private.
支持国服吗？Does it support the China realm?
支持。国服与国际服均可用，界面提供简体中文、繁体中文与英文。国服交易搜索需要在设置中填入有效的会话 Cookie。 Yes. Both China and Global realms are supported, with Simplified Chinese, Traditional Chinese and English UI. China realm trade search requires a valid session cookie in settings.
它是开源的吗？Is it open source?
基于 Tauri v2 + React 19 + Rust 构建。当前源码仓库暂未公开，本页面仅用于介绍与分发。 Built with Tauri v2 + React 19 + Rust. The source repository is currently private; this page is for introduction and distribution only.
